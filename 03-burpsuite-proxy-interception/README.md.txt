# Lab 03 — Burp Suite Proxy Interception

**Tool:** Burp Suite Community Edition v2025.10.6  
**Environment:** Kali Linux VM (isolated lab)  
**Date:** May 2026

## Summary
Configured Burp Suite as an intercepting HTTP proxy, routed 
Firefox traffic through it, installed the PortSwigger CA 
certificate, and captured a plaintext credential POST submission.

## Skills Demonstrated
- HTTP proxy configuration
- CA certificate installation and MITM concepts
- HTTP request inspection and analysis
- Web application penetration testing fundamentals
